# PSK 解密工具使用说明

## 简介

`decrypt_psk.py` 是一个用于解密使用 XOR 加密的字符串的 Python 工具。

## 加密算法说明

### 加密流程
1. **十六进制字符串 → 字节数组**：将十六进制字符串转换为字节数组（每2个字符 = 1字节）
2. **异或加密**：每个字节与密钥 `0xAA` 进行异或运算
3. **字节数组 → 十六进制字符串**：将加密后的字节数组转换回十六进制字符串

### 解密流程（反向操作）
1. **十六进制字符串 → 字节数组**：将加密的十六进制字符串转换为字节数组
2. **异或解密**：每个字节与密钥 `0xAA` 进行异或运算（异或是自逆的）
3. **字节数组 → 十六进制字符串**：将解密后的字节数组转换回十六进制字符串

## 使用方法

### 基本用法

```bash
# 解密指定的加密字符串
python3 decrypt_psk.py '185d0de99d45ed33d12b19afdef15c84'
```

### 在 Python 代码中使用

```python
from decrypt_psk import decrypt_psk

# 解密加密的 PSK 字符串
encrypted = '185d0de99d45ed33d12b19afdef15c84'
decrypted = decrypt_psk(encrypted)
print(f"解密结果: {decrypted}")
```

## 输入格式

- **输入**：十六进制字符串（只包含字符 `0-9`, `a-f`, `A-F`）
- **长度**：可以是任意长度（偶数或奇数）
- **示例**：`185d0de99d45ed33d12b19afdef15c84`

## 输出格式

- **输出**：解密后的十六进制字符串（小写）
- **长度**：与输入字符串长度相同
- **示例**：`b2f7a74337ef47997b81b305745bf62e`

## 注意事项

1. **密钥固定**：当前使用的加密密钥是 `0xAA`，与代码中的 `BK_WIFI_ENCRYPT_KEY` 保持一致
2. **字符串格式**：输入必须是有效的十六进制字符串
3. **Python 版本**：需要 Python 3.x
4. **字符大小写**：工具会自动处理大小写，输出统一为小写

## 技术细节

### 加密密钥
- 密钥值：`0xAA` (170)
- 定义位置：代码中 `BK_WIFI_ENCRYPT_KEY = 0xAA`

### 算法特点
- **对称加密**：加密和解密使用相同的密钥和算法
- **异或运算**：异或运算是自逆的，`(A ^ B) ^ B = A`
- **字节级加密**：以字节为单位进行加密，保持数据长度不变

## 故障排除

### 问题：解密结果不正确
- **检查输入格式**：确保输入是有效的十六进制字符串
- **检查密钥**：确认使用的密钥是 `0xAA`
- **检查长度**：确保输入字符串长度正确

### 问题：Python 版本错误
```bash
# 检查 Python 版本
python3 --version
# 需要 Python 3.x
```
